# EXPLORATORY-DATA-ANLYSIS-USING-PYTHON
## 📊 Exploratory Data Analysis — Amazon Sales Dataset

### Overview
This project performs a full Exploratory Data Analysis (EDA) on an Amazon 
sales dataset containing 50,000 transaction records spanning 2022–2023.

### Dataset
- 50,000 transactions | 13 variables
- 6 Product Categories: Electronics, Fashion, Books, Beauty, Sports, Home & Kitchen
- 4 Regions: North America, Asia, Europe, Middle East
- 5 Payment Methods: UPI, Credit Card, Debit Card, Wallet, Cash on Delivery

### Key Findings
- Price is the strongest revenue driver (r = 0.71)
- Quantity sold has a moderate positive effect (r = 0.59)
- Discounts have a slight negative impact on revenue (r = -0.14)
- Product category, region, and payment method show negligible variation in sales outcomes

### Tools & Libraries
Python | pandas | matplotlib | seaborn | Jupyter Notebook

### Analysis Includes
- Univariate Analysis: Histograms, KDE curves, Box plots, Skewness charts,
  Count plots, Pie charts, Frequency bar charts
- Bivariate Analysis: Correlation heatmap, Scatter plots, Box plots by 
  category, Mean bar plots, Violin plots, Pair plot
- 21 visualisations total
